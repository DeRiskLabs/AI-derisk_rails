# Derisk Rails Skills

This repository contains Derisk Labs Rails-general skills.

Gemini does not automatically invoke `SKILL.md` files the same way Claude Code and Codex do. When this collection is available in a workspace:

1. Read `INDEX.md`.
2. Select the smallest relevant skill directory.
3. Read that skill's `SKILL.md` before acting.
4. Follow `references/` only when the skill says the reference is needed.

Use these skills for Rails architecture, framework-edge adapters, models, forms, jobs, mailers, serializers, validators, I18n copy, Slim views, view models, presenters, stylesheets, and Rails tests. Do not apply Layers-specific rules unless the Derisk Layers skills are also installed and selected.
